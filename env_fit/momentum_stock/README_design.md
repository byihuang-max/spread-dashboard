# 强势股环境诊断 — 设计文档 v1

> 2026-03-01 Roni × Eva 共创

## 核心命题

**强势股/打板策略超额好不好做？** — 不看策略P&L，看短线博弈环境质量。

## 三层框架

### ① 赚钱效应（温度）
- 晋级率（1进2、2进3）+ 连板高度（空间天花板）
- 炸板率取反（封板可信度）
- 封板资金级别：100亿+市值涨停数、300亿+市值涨停数
- 题材级别量化：300亿+市值涨停作为锚点信号——大市值封板过滤噪音，不靠主观判断
- 人工优先级参考：绝对创新 > 政策催化 > 财务超预期 > 冷门题材

### ② 板块微观结构（方向与质量）
- 300亿+涨停票 → 识别主攻行业
- 细分行业（申万）：涨跌家数比、成交额、平均涨幅
- 100亿+涨停按行业分布
- 上中下游产业链传导验证：全链联动 = 大题材确认，龙头独舞 = 独立事件
- 借鉴中观景气度传导框架

### ③ 情绪预警（条件链验证）
- 信号层：高位断板、炸板率飙升、晋级率断崖
- 确认层：全A成交额趋势（方向 + 变异系数CV）
- 过滤层：耐心资本变化率（patient_capital模块，看时序斜率不看截面）
- 判定：信号+确认+无托底 → 🔴红灯；信号+确认+有托底 → 🟡黄灯；仅信号 → 观察

## 关键投研洞察（Roni）
- 情绪先于大盘：退潮往往领先指数下跌
- 但情绪枯竭 ≠ 必然下跌：耐心资本会托底
- 市场4类定价力量：游资、公募/保险、国家队、量化
- 耐心资本出信号要看时序变化率，截面绝对值没意义
- 全A成交额是确认条件，不是独立因子
- 300亿+涨停 = 大题材量化代理

## 嵌套逻辑

```
流动性隐含在①②中（资金级别+板块成交额）

赚钱效应 × 板块结构 → 环境质量
情绪预警条件链 → 风险灯
```
