# 涨跌停封单额每日轧差 (Seal Spread)

## 指标定义

**封单额轧差** = 当日全市场涨停封单额合计 − 跌停封单额合计

- **正值大** → 多头封锁力量强，市场亢奋，资金在追涨停
- **负值大** → 空头封锁力量强，恐慌蔓延，资金在砸跌停
- **接近零** → 多空均衡

## 核心用法（抄底指标）

- 封单额轧差大幅为负 → 跌停远多于涨停 → 恐慌到极致 → **潜在底部信号**
- 可做分位数排名（近 1 年/2 年），轧差落入 <5% 分位数 → 强底部信号
- 配合涨停指数 BIAS、涨跌家数等共振验证

## 数据源

- 来自上两级 `_cache/{date}.json` 中的 `U`（涨停）和 `D`（跌停）
- 每只股票的 `fd_amount` 字段 = 封单额（元）
- `limit_amount` 也可参考（有时为 null，fd_amount 更稳定）

## 输出字段设计

| 字段 | 说明 |
|------|------|
| date | 交易日 |
| up_count | 涨停家数 |
| up_seal_total | 涨停封单额合计（亿元） |
| down_count | 跌停家数 |
| down_seal_total | 跌停封单额合计（亿元） |
| seal_spread | 轧差 = up_seal_total − down_seal_total（亿元） |
| seal_ratio | 涨停封单额 / (涨停+跌停封单额)，0~1 |
| spread_ma5 | 轧差 5 日均线 |
| spread_ma20 | 轧差 20 日均线 |
| spread_pct_1y | 轧差在近 250 个交易日的分位数 |
| signal | 信号标签（极度恐慌/偏空/均衡/偏多/极度亢奋） |

## 信号阈值（初版方案）

| 分位数区间 | 信号 | 含义 |
|-----------|------|------|
| <5% | 🟢 极度恐慌（抄底） | 跌停封单远超涨停，恐慌见底 |
| 5%~20% | 🔵 偏空 | 空头占优但未到极端 |
| 20%~80% | ⚪ 均衡 | 正常波动 |
| 80%~95% | 🟡 偏多 | 多头占优 |
| >95% | 🔴 极度亢奋 | 涨停封单远超跌停，过热警告 |
