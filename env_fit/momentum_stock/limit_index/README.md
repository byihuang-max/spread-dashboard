# limit_index — 涨停指数（昨日涨停 / 昨日首板）

## 概述

构建「昨日涨停指数」和「昨日首板指数」的净值曲线，用于衡量追涨停策略的拥挤度和盈亏效率。

**核心思路：** 假设 T-1 日收盘时等权买入所有涨停（或首板）股票，T 日收盘卖出，计算当日组合收益率，逐日累计成净值曲线。

## 核心指标

| 指标 | 公式 / 说明 |
|------|-------------|
| **净值曲线** | 逐日累计收益，长期应斜向上（涨停正期望） |
| **BIAS（乖离率）** | `(净值 - MA20) / MA20 × 100`<br>过高 = 涨停股被过度追捧（拥挤）<br>过低 = 情绪冰点（潜在反转） |
| **高低开** | `avg(open - pre_close) / pre_close × 100`<br>隔夜溢价/折价，反映市场对涨停股的预期 |

## 信号灯 v3：三重交叉验证

旧版用 BIAS 固定 ±3% 阈值判信号，但涨停指数是长期正期望曲线，BIAS 几乎永远 >3%，导致信号一直喊"过度拥挤"，失去预警价值。

### v3 设计（三因子交叉）

| 因子 | 逻辑 | 说明 |
|------|------|------|
| ① 净值回撤/上行 | 当前净值 vs 近20日峰值的回撤幅度 | 直接衡量"涨停策略最近是赚是亏" |
| ② BIAS 变化方向 | 近3日均 vs 前3日均的差值（BIASΔ） | 看 BIAS 的方向而非绝对值 |
| ③ 情绪周期标签 | 来自合成情绪指数的周期判定 | 退潮/冰点/回暖/加速/分歧/震荡 |

### 信号判定规则

三因子方向一致 = ★★★（高置信），两因子一致 = ★★，矛盾 = 分歧观望。

| 信号 | 触发条件 |
|------|----------|
| 🔵 快速降温 ★★★ | 净值回撤 >3% + BIAS 下行 + 情绪退潮/冰点 |
| 🔵 降温 ★★★ | 净值回撤 + BIAS 下行 + 情绪退潮/冰点 |
| 🔵 全面降温 ★★★ | 全板 + 首板 BIAS 同步下行 + 情绪退潮/冰点 |
| ⬇️ 边际降温 ★★ | 净值回撤 + BIAS 下行，情绪未同步 |
| ⬇️ 情绪降温 ★★ | 情绪退潮 + BIAS 下行 |
| 🟡 见顶退潮 ★★★ | 净值高位 + BIAS 掉头 + 情绪退潮 |
| 🟡 见顶分歧 ★★ | 净值高位 + BIAS 掉头 |
| 🟡 多空分歧 | 涨停指数与情绪信号矛盾 |
| 🟢 筑底回暖 ★★★ | 净值止跌 + BIAS 回升 + 情绪回暖 |
| 🟢 筑底回暖 ★★ | 净值止跌 + BIAS 回升 |
| 🔴 加速升温 ★★★ | 净值连涨 + BIAS 加速 + 情绪加速，警惕过热 |

### 数据验证（120日回测）

- 退潮日：BIASΔ<0 占 75%，平均收益仅 +0.50%
- 加速日：BIASΔ>0 占 90%，平均收益 +3.15%
- 情绪周期与涨停指数高度共振，交叉验证有效

## 文件说明

| 文件 | 用途 |
|------|------|
| `limit_index_data.py` | 数据拉取 + 净值/BIAS 计算（CSV 增量模式） |
| `limit_index.csv` | 完整历史数据（date, nav, bias, return, gap 等） |
| `limit_index.json` | JSON 格式输出（供前端 inject_momentum.py 读取） |
| `_cache_daily/` | 全市场日线行情缓存（按日期，避免重复拉取） |

## 数据流

```
上级 _cache/{date}.json（涨停股代码，来自 momentum_data.py）
  ↓ limit_index_data.py
  ├── 读取 T-1 日涨停股代码
  ├── 拉取 T 日行情（Tushare daily API）
  ├── 计算等权收益率 + 高低开
  └── 累计净值 + 计算 BIAS
  ↓
limit_index.csv / limit_index.json
  ↓ inject_momentum.py（在上级目录）
index.html（强势股 tab → 涨停指数图表 + 信号灯）
```

## CSV 字段

| 字段 | 说明 |
|------|------|
| `date` | 收益计算日（T 日） |
| `limit_date` | 涨停日（T-1 日，即买入信号日） |
| `all_count` / `all_valid` | 全涨停股票总数 / 有效匹配数 |
| `all_return` | 全涨停等权收益率 (%) |
| `all_gap` | 全涨停平均高低开 (%) |
| `all_nav` | 全涨停累计净值 |
| `all_bias` | 全涨停 BIAS (MA20) |
| `first_*` | 首板指数对应字段（同上） |

## 数据源

- **涨停股代码：** 上级 `_cache/` 目录（由 `momentum_data.py` 拉取 Tushare `limit_list_d` 生成）
- **日线行情：** Tushare Pro `daily` 接口（open / close / pre_close / pct_chg）

## 历史

- **初版（v1/v2）：** BIAS 固定 ±3% 阈值，涨停指数长期正期望导致 BIAS 永远超标，信号失效
- **v3（2026-03-08）：** 改为三重交叉验证（净值回撤 × BIAS方向 × 情绪周期），解决误报问题
