# 强势股择时研究初衷与交易指导定位

- 日期：2026-03-12
- 类型：决策 / 研究定位
- 标签：择时, 强势股, 冰点, 交易指导, 研究框架

## 核心背景
Roni 明确补充了这条择时研究线的真实初衷：

1. 起点不是为了做一个“好看的量化报告”，而是为了**指导交易**。
2. 研究源头来自 Roni 自己对市场的理解：先构建 FOF 看板里的**强势股板块**，再把围绕强势股生态能想到的择时思路拆成**单因子**。
3. 单因子研究之后，才进一步形成**双因子冰点**，再往后才有风格过滤与策略优化。
4. 整条链路应理解为：
   **交易认知 → 强势股观察框架 → 单因子测算 → 双因子冰点状态 → 风格过滤 / 优化 → 交易动作映射**。

## 更深一层的初衷
Roni 提到一个关键动机：

- 他的朋友在做**择时量化交易**，但对方的择时模块写得比较一般。
- 对方的思路更偏向：
  - 用阿尔法强弱分阈值
  - 做 15min 截面
  - 打分高的票多，就把仓位加上去
- Roni 希望从自己更强的**市场逻辑与交易理解**出发，给这类纯量化择时框架补充一个新视角。

也就是说，这套研究不是为了替代量化执行，而是为了给量化执行增加一个**更有市场语义、更有逻辑约束**的上层框架。

## Roni 的自我定位
Roni 明确说：

- **对市场有自己的理解**
- 但**不擅长量化**

这意味着协作分工应当是：

- Roni 负责：市场理解、交易逻辑、状态定义、框架约束
- AI / 量化实现负责：把这些认知翻译成可测算、可回测、可落地的研究结构

## 对报告结构的启发
因此，择时总报告的主叙事不应只写成“V1 / V1.1 / V1.2”。
更准确的表达应该是：

1. 为什么从强势股生态出发
2. 如何把交易认知拆成单因子
3. 为什么单因子进一步抽象成双因子冰点
4. 为什么冰点之后还需要风格过滤与动作映射
5. 最终怎样服务于交易指导

## 结论
这条线的本质不是“为了找到最优参数”，而是：

> 把 Roni 的市场理解，转成一套能辅助交易、能补强纯量化择时视角的研究框架。
