# iFinD MCP 金融数据接入

- 日期：2026-03-26
- 类型：工具接入
- 标签：iFinD, MCP, 数据源, 同花顺, API

## 背景
同花顺于 2026-03-12 正式上线 iFinD MCP 金融数据服务，专为 AI 智能体设计，支持自然语言查询金融数据。

## 接入的4个服务

### 1. 股票服务 (stock)
- 智能选股（自然语言条件筛选）
- 股票摘要、基本资料、行情与技术指标
- 财务数据、股东结构、风险指标、ESG、重大事件
- 示例：`"电子行业市值大于100亿的股票"`

### 2. 基金服务 (fund)
- 基金搜索、基本资料、行情业绩
- 持仓明细、持有人结构、财务指标、基金公司信息
- 对 FOF 工作直接有用
- 示例：`"南方基金新能源ETF"`

### 3. 宏观经济服务 (edb)
- GDP、CPI、PPI、行业经济指标、大宗商品
- 支持"先搜再查"：先 search_edb 模糊搜索，再 get_edb_data 精确取数
- 示例：`"光伏电池产量202301-202506"`

### 4. 新闻公告服务 (news)
- 财经新闻语义检索、上市公司公告搜索、热点事件
- 示例：`"脑机接口技术最新进展"`

## 技术细节
- 协议：MCP (Model Context Protocol) over Streamable HTTP
- 端点：`https://api-mcp.51ifind.com:8643/ds-mcp-servers/`
- 认证：JWE Authorization token
- 直连可用，不需要代理
- Skill 位置：`~/.openclaw/extensions/ifind-finance-data/`
- 调用方式：Node.js（推荐，无依赖）或 Python

## 与现有 iFinD HTTP API 的区别
- HTTP API：需要知道具体指标代码和字段名，写代码拼参数
- MCP 方式：自然语言查询，AI 自动理解意图并返回结构化结果
- 两套可以并存，MCP 更适合 AI agent 交互场景

## 对 GAMT 看板的价值
- 基金数据：Tushare 覆盖不到的公募基金详细数据
- 宏观指标：可替代部分 iFind HTTP API 的宏观数据拉取
- 新闻公告：叙事监控模块可用于热点事件追踪
