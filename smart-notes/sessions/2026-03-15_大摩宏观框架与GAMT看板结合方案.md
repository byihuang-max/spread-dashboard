# 大摩宏观框架与GAMT看板结合方案

**日期：** 2026-03-15  
**类型：** 对话记录 - 框架设计  
**关键词：** 大摩四象限、宏观逻辑链、资产配置、利润周期、内需接棒

---

## 一、背景

Roni 提供了大摩李圣尧的宏观交易框架PDF，核心理念：
- **经济范式变更：** 旧范式（货币紧缩+全球化）→ 新范式（财政主导+通胀回归）
- **四象限驱动：** 增长/通胀/货币/信用 → 标准化映射 → 资产配置（股/债/商品）
- **风险预算控制：** 通过组合波动率预算控制风险，而非单纯看资产波动率
- **多资产组合：** 股/债/商品底层现金流结构不同，提供离散收益

**GAMT现状：**
- ✅ 宏观数据丰富（PMI/CPI/PPI/流动性/利率汇率）
- ✅ 策略环境适配（量化股票/强势股/CTA/转债/套利）
- ❌ **缺少决策逻辑：** 数据 → 象限定位 → 资产配置建议

---

## 二、核心逻辑链（优先级排序）

### 逻辑链1：增长-通胀四象限 → 资产配置决策（最高优先级）

**框架：**
```
宏观四维度（增长/通胀/货币/信用）
    ↓ 标准化打分（Z-score）
四象限定位（增长↑↓ × 通胀↑↓）
    ↓ 映射规则
资产配置权重（股/债/商品/现金）
    ↓ 细化到策略
策略环境适配（量化股票/强势股/CTA/转债/套利）
```

**数据需求（P0）：**
- 增长：PMI（已有）、社融结构（需补充）
- 通胀：CPI/PPI（已有）、PPI环比（需补充）
- 货币：Shibor/DR007（已有）、R007-DR007利差（需补充）
- 信用：企业债利差（需补充）、社融信贷占比（需补充）

**和GAMT结合：**
- 新增**四象限仪表盘**（宏观层顶部）
- 输出：当前象限 + 资产配置建议 + 策略权重建议
- 美林时钟作为验证（四象限 vs 美林时钟一致性）

---

### 逻辑链2：中国利润周期 → 股票策略择时（第二优先级）

**框架：**
```
PPI环比转正（价格止跌）
    ↓
出厂价-购进价剪刀差为正（企业可以转嫁成本）
    ↓
PMI新订单/出货改善（需求回暖）
    ↓
规上工业企业利润同比转正（利润上行段开始）
    ↓
股票策略环境改善（量化股票/强势股适配度↑）
```

**数据需求（P0）：**
- PPI环比（Tushare已有）
- 出厂价-购进价剪刀差（Tushare已有）
- 规上工业企业利润（Tushare已有）
- PMI新订单（已有）

**和GAMT结合：**
- 新增**利润周期监控模块**（中观层）
- 输出：利润周期阶段（筑底/上行/见顶/下行）+ 股票策略适配度
- 作为策略环境适配的输入信号

---

### 逻辑链3：中国内需接棒验证 → 风格切换信号（第三优先级）

**框架：**
```
社融结构：政府债占比↓ + 居民/企业信贷占比↑（内生需求接棒）
    ↓
居民中长期贷款回升（购房/消费意愿↑）
    ↓
核心CPI抬升（需求驱动通胀）
    ↓
进口资本品走强（企业扩产）
    ↓
内需接棒确认 → 大盘价值风格受益（银行/地产/消费）
```

**数据需求（P0）：**
- 社融结构（Tushare `cn_m`表）
- 居民中长期贷款（Tushare已有）
- 核心CPI（Tushare已有）
- 进口资本品（海关/iFind，需补充）

**和GAMT结合：**
- 新增**内需接棒验证模块**（宏观层）
- 输出：内需接棒进度（0-100%）+ 风格建议（大盘价值 vs 小盘成长）
- 作为风格轧差的输入信号

---

### 逻辑链4：美国软着陆 vs 硬着陆 → 全球风险开关（第四优先级）

**框架：**
```
【软着陆路径】
服务业PMI不塌（>50）+ 红皮书零售正增长 + 工资温和回落
    ↓
核心PCE接近2%（通胀可控）
    ↓
美联储降息 → 全球风险偏好改善 → A股受益

【硬着陆路径】
FCI持续收紧 + 信用利差扩张 + 消费断崖
    ↓
美国衰退 → 全球风险厌恶 → A股避险（转债/套利策略受益）
```

**数据需求（P1）：**
- 服务业PMI（需爬虫）
- 红皮书零售（需爬虫）
- 核心PCE（FRED免费）
- 信用利差（FRED免费）
- 金融条件指数（高盛FCI或自建）

**和GAMT结合：**
- 新增**美国软硬着陆监控模块**（全球层）
- 输出：软着陆概率（0-100%）+ 全球风险开关（risk-on/off）
- 作为红灯预警的全球风险维度

---

### 逻辑链5：经济范式转换 → 战略资产配置调整（长期战略）

**框架：**
```
【旧范式（1990-2020）】
货币紧缩 + 全球化 + 通胀目标制
    ↓
股债60/40组合有效 + 债券是避险资产

【新范式（2020-现在）】
财政主导 + 逆全球化 + 通胀回归
    ↓
股债相关性上升（同时下跌）
    ↓
商品/黄金成为避险资产 + 多资产组合必要性提升
```

**数据需求（P2）：**
- 财政赤字率（Tushare/财政部）
- 全球化指数（贸易依存度，需自建）
- 通胀预期锚定程度（5y5y TIPS breakeven，FRED）

**和GAMT结合：**
- 新增**经济范式监控模块**（战略层）
- 输出：当前范式（旧/新/转换中）+ 战略资产配置建议
- 结合宏观Meme叙事（地缘风险/去美元化/财政主导）

## 三、大摩指标体系与逻辑演绎

### 美国宏观

#### 增长维度
1. **服务业PMI（整体+新订单+就业）**  
   逻辑：美国70%是服务业 → 新订单↑+就业↑ = 服务业自我强化（软着陆）

2. **红皮书商业零售销售（周同比）**  
   逻辑：消费占GDP 68% → 周度高频领先指标 → 断崖式下滑 = 硬着陆第一预警

3. **ISM制造业PMI**  
   逻辑：vs S&P500 EPS → 判断盈利驱动来源

#### 通胀维度
4. **核心CPI/核心PCE**  
   逻辑：剔除食品能源 → 反映需求驱动通胀 → 核心PCE>2.5% = 美联储不敢降息

5. **CPI住房分项（OER/租金）**  
   逻辑：占核心CPI权重40% → 通胀"最后一公里" → OER滞后实际租金12-18个月

6. **Realtor.com新签租金**  
   逻辑：领先实际租金6-12个月 → 新签租金↓ → 实际租金↓ → OER↓ → 核心CPI↓

7. **ISM服务业物价指数**  
   逻辑：服务通胀扩散指标 → >60 = 通胀扩散，<50 = 通胀收敛

8. **核心服务通胀（住房外服务）**  
   逻辑：核心CPI - 住房 - 商品 → 核心服务>4% = 美联储不敢降息（工资粘性）

#### 工资与成本
9. **私营部门名义工资增速（尤其服务业）**  
   逻辑：工资↑ → 企业成本↑ → 服务价格↑ → 核心CPI↑ → 工资>4% = 通胀粘性强

10. **单位劳动力成本（ULC）**  
    逻辑：ULC = 工资增速 - 生产率增速 → ULC>0 = 通胀压力，ULC<0 = 通胀可控

#### 金融条件与资金流
11. **金融条件指数（FCI）**  
    逻辑：利率+信用利差+股市+汇率 → FCI宽松 = 增长加速，FCI收紧 = 增长放缓

12. **信用利差**  
    逻辑：企业债收益率 - 国债收益率 → 利差扩张 = 经济衰退预警

13. **美国财政部TIC数据**  
    逻辑：海外买入美债 → 长端利率下行 → 美元走强

14. **经济意外指数**  
    逻辑：实际数据 - 市场预期 → 指数>0 = 数据超预期 → 股市上涨/美联储鹰派

#### 利率与政策
15. **联邦基金利率/政策利率路径**  
    逻辑：降息 → 融资成本下降 → 股市上涨/美元走弱

16. **10Y美债收益率**  
    逻辑：无风险利率基准 → 10Y上行 = 股市估值压缩

17. **期限溢价（Term Premium）**  
    逻辑：10Y美债 - 预期短期利率 → 期限溢价>0 = 投资者要求补偿（通胀/财政风险）

18. **收益率曲线形态（2-5Y vs 10Y）**  
    逻辑：熊陡 = 通胀预期上升，牛陡 = 降息预期，倒挂 = 衰退预警

19. **财政供给节奏（国债发行）**  
    逻辑：发债增加 → 供给压力 → 长端利率上行

20. **实际利率（TIPS）**  
    逻辑：名义利率 - 通胀预期 → 实际利率上行 = 黄金下跌/久期资产下跌

#### 大宗与避险
21. **5y5y通胀预期（TIPS breakeven）**  
    逻辑：5年后的5年通胀预期 → 5y5y>2.5% = 通胀预期脱锚

22. **汽油/原油价格**  
    逻辑：供给扰动 = 油价上涨 → CPI上行，需求冲击 = 油价下跌 → CPI下行

23. **Brent原油+裂解价差+库存节奏**  
    逻辑：库存下降+裂解价差扩大 = 需求强劲 → 油价上涨

24. **黄金**  
    逻辑：地缘风险溢价 + 实际利率 + 央行买盘 → 实际利率下行 = 黄金上涨

25. **工业金属库存+近月升水**  
    逻辑：库存下降+近月升水 = 供给紧张 → 价格弹性大

### 中国宏观

#### 信用与内需
26. **社融总量+结构拆分**  
    逻辑：政府债占比高 = 财政发力但内生需求弱，居民/企业信贷占比高 = 内需接棒

27. **居民中长期贷款**  
    逻辑：中长贷↑ = 购房/消费意愿强 → 内需扩张

28. **金融机构存贷款增速差**  
    逻辑：增速差扩大 = "钱多但不敢花" → 流动性陷阱

29. **社零（服务vs商品、必选vs可选）**  
    逻辑：服务消费↑ = 消费升级，可选消费↑ = 消费信心强

30. **固定资产投资（制造业/基建/地产）**  
    逻辑：制造业投资↑ = 企业扩产 → 经济内生动力强

31. **GDP需求侧拆分**  
    逻辑：消费贡献↑ = 内需接棒，净出口贡献↓ = 外需走弱

#### 通胀与利润链条
32. **CPI（整体+食品vs核心服务）**  
    逻辑：食品CPI↑ = 猪周期/供给冲击，核心服务CPI↑ = 就业收入预期改善

33. **核心CPI**  
    逻辑：CPI - 食品 - 能源 → 核心CPI↑ = 需求驱动通胀

34. **PPI（整体+分行业）**  
    逻辑：PPI环比转正 = 利润上行段开始

35. **PPI环比**  
    逻辑：月度变化 → 环比转正 = 价格止跌 → 利润改善关键信号

36. **出厂价-购进价剪刀差**  
    逻辑：剪刀差>0 = 企业可以转嫁成本 → 利润改善

37. **规上工业企业利润**  
    逻辑：利润同比转正 = 盈利止跌

38. **主营业务收入利润率+资产负债率**  
    逻辑：利润率↑+资产负债率↓ = 企业健康度改善

#### 外需与汇率
39. **出口数量指数vs出口价格指数**  
    逻辑：数量↑价格↓ = "以价换量"，数量↑价格↑ = 外需强劲

40. **PMI新出口订单**  
    逻辑：领先指标 → 新出口订单>50 = 外需扩张

41. **进出口金额分项**  
    逻辑：资本品进口↑ = 企业扩产 → 内需扩张

42. **银行代客结售汇/净结汇**  
    逻辑：净结汇↑ = 人民币升值压力

43. **USDCNY即期+中间价Fixing Bias**  
    逻辑：Bias>0 = 央行引导贬值，Bias<0 = 央行引导升值

44. **USDCNY FX Swap Points/CIP Basis**  
    逻辑：CIP Basis>0 = 离岸人民币流动性紧张 → 人民币升值压力

45. **美元指数**  
    逻辑：美元指数↑ = 人民币被动贬值压力

#### 市场与资金流
46. **金融条件指数（中国）**  
    逻辑：FCI宽松 = 风险偏好改善 → 股市上涨

47. **沪深300**  
    逻辑：A股基准指数

48. **南向资金净流入**  
    逻辑：南向资金↑ = 港股边际定价者 → 港股上涨

49. **收益率曲线（中国国债）**  
    逻辑：陡峭化 = 经济复苏预期 → 长端利率上行

## 六、关键洞察

1. **大摩框架的核心价值：** 把宏观数据标准化为四象限 → 映射到资产配置决策，填补了"数据丰富但缺少决策逻辑"的空白

2. **共振信号的重要性：** 不依赖单一指标，而是看多指标共振（例如：PPI环比转正 + 剪刀差为正 + 订单改善 = 利润上行段确认）

3. **经济范式转换：** 旧范式（货币紧缩+全球化）→ 新范式（财政主导+通胀回归），多资产组合（股/债/商品）必要性提升

4. **逻辑链的层次：**
   - 战略层：经济范式监控（长期）
   - 战术层：四象限定位 → 资产配置（中期）
   - 交易层：利润周期/内需接棒 → 策略择时（短期）

5. **GAMT看板的演进方向：**
   - 从"数据展示"升级为"决策支持"
   - 从"单一指标"升级为"共振信号"
   - 从"静态配置"升级为"动态调整"
