# GPT-Image2 工业级提示词模板体系

> 来源：[awesome-gpt-image-2](https://github.com/freestylefly/awesome-gpt-image-2)
> 338个案例逆向工程，13套工业级模板，核心理念：Prompt as Code

---

## 核心方法论：三步走

1. **定任务类型**：UI、海报、电商、信息图、角色设定、出版物等
2. **锁结构约束**：比例、布局、模块数量、镜头语言、文字要求
3. **补风格材质**：色彩、光线、笔触、氛围、质感

每套模板都有两种格式：常规文本模板（给人用）+ JSON结构化模板（给Agent/脚本调用）

---

## 1. UI与界面

**模板：**
```
为[产品类型]生成一张[平台，如iOS/Android/Web]界面图。
核心功能：[功能点A]、[功能点B]、[功能点C]。
视觉风格：[极简/科技/拟物]，主色[颜色]，强调色[颜色]。
布局：[顶部导航/双栏/卡片流]，信息层级清晰，留白充足。
输出：高保真UI截图，文字清晰可读，比例[9:16/16:9]。
```

**JSON版：**
```json
{
  "type": "UI Screenshot",
  "platform": "iOS",
  "product": "Fitness App",
  "layout": "Card-based feed with bottom tab bar",
  "style": {"theme": "Dark Mode", "primary_color": "Neon Green", "typography": "Clean sans-serif"},
  "content": {"header": "Today's Activity", "cards": [{"title": "Running", "data": "5.2 km", "button": "Start"}]},
  "constraints": "High fidelity, readable text, 9:16 aspect ratio"
}
```

**避坑：** 明确"平台+比例+布局"；强制文字可读，避免乱码火星文。

## 2. 图表与信息可视化

**模板：**
```
生成[主题]信息图，目标读者为[人群]。
结构：标题区 + [3-5]个模块（每模块含图标、短标题、1-2句说明）。
图表类型：[流程图/对比图/关系图/时间线]。
风格：[专业报告/科普插画]，主色[颜色]，背景[浅色/深色]。
输出：信息层级清晰、可读性高的中文信息图。
```

**避坑：** 控制模块数量和图表类型；文案用短句，别塞大段文字。

---

## 3. 海报与排版

**模板：**
```
设计一张[活动/产品/电影]海报，主题为[主题词]。
主视觉：[主体元素]，标题文案：[标题]，副标题：[副标题]。
版式：[居中/左对齐/对角构图]，风格：[复古/未来/极简]。
色彩：[主色+辅色]，氛围：[情绪关键词]。
输出：可用于社媒传播的高分辨率海报。
```

**避坑：** 主视觉说清楚是什么；主标题副标题写死，别让模型自由发挥。

---

## 4. 商品与电商

**模板：**
```
生成[商品名]电商主图，卖点为[卖点1]、[卖点2]。
场景：[纯色棚拍/生活方式场景]，镜头：[特写/半身/全景]。
材质细节：[材质关键词]，灯光：[柔光/侧光/轮廓光]。
附加元素：[价格角标/卖点icon/促销文案]。
输出：电商平台可直接使用的商品展示图。
```

**避坑：** 材质和光影是灵魂，堆叠材质+灯光关键词；促销文案只给1-2句核心的。

---

## 5. 品牌与标志

**模板：**
```
为[品牌名]设计品牌视觉方案。
品牌关键词：[关键词1]、[关键词2]、[关键词3]。
包含：Logo方向[几何/字标/图形]、辅助图形、主辅色、应用示意。
风格：[现代/高级/亲和]，行业：[行业]，受众：[受众]。
输出：统一风格的品牌识别视觉图。
```

**避坑：** 做减法，先定品牌关键词再出视觉；强制纯白背景方便抠图。

---

## 6. 建筑与空间

**模板：**
```
生成[空间类型]设计效果图，功能定位为[用途]。
风格：[现代简约/工业/新中式]，材质：[木/石/金属/玻璃]。
空间结构：[开敞/分区]，动线：[主通道说明]。
光线：[自然采光/人工照明方案]，时间：[白天/夜景]。
输出：写实建筑空间渲染图。
```

**避坑：** 用"Eye-level perspective（人眼视角）"压住透视变形；室内外冷暖对比提升高级感。

---

## 7. 摄影与写实

**模板：**
```
拍摄主题：[人物/物品/街景]，场景为[地点]。
摄影参数风格：[35mm/85mm]，[浅景深/深景深]，[纪实/电影感]。
光线：[自然光/夜景霓虹/逆光]，情绪：[情绪词]。
细节要求：[肤质/材质/颗粒感]。
输出：高写实摄影风格图像。
```

**避坑：** 加瑕疵（毛孔、雀斑、胶片颗粒）提升真实感；用相机参数（f/1.4、50mm）代替模糊描述。

---

## 8. 插画与艺术

**模板：**
```
创作[题材]插画，主角为[角色/主体]。
画风：[日漫/水彩/扁平/厚涂]，线条：[细腻/粗犷]。
配色：[配色方案]，背景：[简洁/复杂场景]。
构图：[近景/中景/远景]，重点表现[细节]。
输出：可用于封面或社媒发布的高质量插画。
```

**避坑：** 锁定笔触（厚涂/水彩）；别直接写大师名，提取大师特征（如"梵高的旋转星空笔触"）。

## 9. 人物与角色

**模板：**
```
设计[角色身份]角色设定图。
外观：[年龄/发型/服饰/配件]，性格：[关键词]。
姿态：[站姿/动态动作]，表情：[情绪]。
世界观：[时代/阵营/职业]，标志性元素：[元素]。
输出：角色主视图 + 风格统一的人设图。
```

**避坑：** 拆解五官（桃花眼、高鼻梁、野生眉）；写清服装材质（丝绸、机能面料）。

---

## 10. 场景与叙事

**模板：**
```
生成[故事主题]场景图，发生在[时间+地点]。
主事件：[事件描述]，主角：[角色]，冲突点：[冲突]。
镜头语言：[广角建立镜头/中景叙事/特写]。
氛围：[紧张/温暖/悬疑]，色调：[冷/暖/高反差]。
输出：具备叙事张力的场景概念图。
```

**避坑：** 必须有动词/事件，别画成风景明信片；用镜头语言（低角度仰拍、倾斜镜头）加戏剧感。

## 11. 历史与古风

**模板：**
```
生成[朝代/古风设定]题材画面，主题为[主题]。
人物：[身份/服饰/器物]，场景：[宫廷/市井/山水]。
美术风格：[工笔/写意/影视写实]，色调：[色调]。
文化细节：[纹样/礼制/建筑要素]。
输出：历史氛围准确的古风题材图。
```

**避坑：** 明确朝代（唐/宋/明）；加"禁用现代元素"防穿帮。

## 12. 文档与出版物

**模板：**
```
制作[文档类型，如菜单/杂志内页/报纸版式]。
版面结构：[栏数/页边距/标题层级]。
内容模块：[封面区/正文区/图表区/脚注]。
字体风格：[衬线/无衬线]，配色：[配色方案]。
输出：可读性强、版式规范的出版物视觉稿。
```

**避坑：** 结构优先（栏数、留白）；正文用模拟文本填充，只写死大标题。

---

## 13. 通用场景

**模板：**
```
任务目标：[你要生成的内容类型]。
输入约束：主体[主体]，场景[场景]，风格[风格]，色彩[配色]。
质量约束：清晰度[高清/4K]，比例[比例]，构图[构图方式]。
输出约束：用于[用途]，需突出[核心信息]。
请输出一版主方案 + 一版备选方案。
```

**避坑：** 先说用途建立上下文；要求一次出主方案+备选方案。

---

## 通用避坑原则

- 文字锁定：所有需要显示的文字必须写死，不让模型自由发挥
- 材质堆叠：材质+光影关键词越具体，画面质感越好
- 参数说话：用相机参数（f/1.4、50mm）代替模糊描述（浅景深、半身照）
- 做减法：约束越精确，输出越可控
- 加瑕疵：适当加入不完美细节（毛孔、颗粒、磨损）提升真实感
