## 背景：当前打分系统架构

GAMT FOF 看板有**两套独立的打分系统**：

### 1. 风险预警系统（alerts_calc.py）
**目标**：识别市场极端风险  
**评分**：0-100分（100=最危险）

**5维度权重**：
- 💧 流动性风险（20%）：DR007/Shibor/R-D价差
- 📊 估值泡沫（25%）：PE分位+股债性价比
- 🔥 情绪过热（20%）：成交额/涨跌停
- 🌍 外部冲击（20%）：中美利差+汇率+期权IV
- 🏃 微观恶化（15%）：北向资金+期权PCR

**输出**：综合风险分数 + 风险等级（🟢🟡🟠🔴）+ 预警文本

### 2. 宏观打分系统（macro_score.py）
**目标**：评估宏观环境+策略适配度+配置建议  
**评分**：0-100分（100=最好）

**5维度权重**：
- 流动性（20%）：Shibor趋势+M1-M2剪刀差
- 基本面（25%）：PMI+美林时钟+CPI-PPI
- 利率汇率（20%）：中美利差+汇率压力+中债走势
- 期权情绪（20%）：IV分位+PCR
- 资金流+拥挤度（15%）：三路资金共识+拥挤度信号

**输出**：宏观总分 + 阶段判断（进攻/均衡/谨慎/防守）+ 策略适配度 + 配置建议

---

## 核心问题诊断

### 问题1：风险预警缺少"市场结构性风险"维度
**表现**：
- 2021年初GameStop、2024年AI泡沫等Meme行情，传统指标（估值/流动性/情绪）可能正常
- 但市场已进入"叙事驱动+反身性高峰"状态，尾部风险极高
- 当前系统无法识别这类结构性风险

**根因**：缺少对"叙事联动+量能加速度"的监控

### 问题2：宏观打分缺少"主力持仓成本"信息
**表现**：
- 指数远低于耐心资本成本 → 主力被套 → 反弹压力大（应加分）
- 指数远高于成本 → 主力获利丰厚 → 抛压风险（应减分）
- 当前系统只看资金流方向，不看持仓成本

**根因**：未整合 patient_capital 模块的成本数据

### 问题3：策略适配度缺少"风格轮动时机"判断
**表现**：
- 风格轧差到极值（如中证2000-沪深300 Z-score > 2）→ 均值回归机会
- 当前系统只看因子表现，不看轮动时机

**根因**：未整合 size_spread 模块的轧差极值信号

### 问题4：宏观打分只看绝对水平，缺少趋势持续性
**表现**：
- PMI连续3个月上行 vs PMI单月反弹，前者信号更强
- 当前系统对两者评分相同

**根因**：缺少趋势斜率计算

### 问题5：风险预警用绝对阈值，缺少历史分位
**表现**：
- DR007>2.5 触发预警，但如果过去1年经常>2.5，这次可能不算异常
- 绝对阈值容易"狼来了"

**根因**：缺少动态阈值机制

---

## 优化方案：三个优先级

### 优先级1：立即可用（数据已有，逻辑成熟）

#### 1.1 Meme反身性信号 → 风险预警系统

**数据来源**：`meme/antifragile/meme_signal.json`

**核心指标**：
- **叙事联动指数（NLI）**：多资产30日滚动相关性均值（0-1）
- **量能加速度（VA）**：近5日均量/近30日均量（BTC35% + 纳指25% + 恒科20% + 科创20%）
- **综合Meme信号**：NLI历史分位×50% + VA历史分位×50%（0-100分）

**整合方案**：
```python
# 在 alerts_calc.py 增加第6维度
# ═══════ 6. 市场结构风险（Meme反身性）═══════
meme = load_json(os.path.join(BASE, 'meme', 'antifragile', 'meme_signal.json'))
if meme:
    current = meme.get('current', {})
    meme_score = current.get('meme_score', 50)
    
    struct['items'].append(f"Meme信号: {meme_score:.0f}/100")
    
    if meme_score >= 75:
        struct['score'] += 40
        result['alerts'].append('🎭 反身性高峰期，叙事驱动达到极致，注意循环破裂')
    elif meme_score >= 50:
        struct['score'] += 20
    
    # 量能加速度作为辅助判断
    va = current.get('va')
    if va and va < 0 and meme_score > 60:
        struct['score'] += 20
        result['alerts'].append('⚠️ 量能萎缩但价格仍高，顶部特征明显')
```

**权重建议**：10%（新增维度，其他维度按比例调整为：流动性18%、估值22%、情绪18%、外部18%、微观14%）

**预期效果**：
- 识别2021年初GameStop式Meme行情
- 识别2024年AI泡沫式反身性高峰
- 提前预警"量能萎缩+价格高位"的顶部特征

#### 1.2 耐心资本成本偏离 → 宏观打分系统

**数据来源**：`micro_flow/patient_capital/patient_capital.json`

**核心指标**：
- 各指数的累计持仓、成本价、浮盈亏（pnl%）
- 重点看沪深300和中证1000（代表大小盘）

**整合方案**：
```python
# 在 macro_score.py 的 score_crowding_flow() 中增加
def score_crowding_flow(d):
    score = 50
    signals = []
    # ... 原有逻辑 ...
    
    # 新增：耐心资本成本偏离
    patient = load("micro_flow/patient_capital/patient_capital.json")
    if patient:
        indices = patient.get('indices', {})
        for idx_name in ['沪深300', '中证1000']:
            idx_data = indices.get(idx_name, {})
            latest = idx_data.get('latest', {})
            if latest:
                pnl = latest.get('pnl', 0)
                
                if pnl < -10:  # 主力被套超过10%
                    score += 10  # 反弹动力强
                    signals.append(f"{idx_name}耐心资本被套{pnl:.1f}%，反弹空间大 🟢")
                elif pnl > 30:  # 主力获利超过30%
                    score -= 10  # 抛压风险
                    signals.append(f"{idx_name}耐心资本获利{pnl:.1f}%，抛压风险 🔴")
    
    return {"score": clamp(score), "signals": signals}
```

**预期效果**：
- 识别主力被套时的反弹机会
- 识别主力获利丰厚时的抛压风险
- 提供更精准的资金流判断

### 优先级2：需要增强（逻辑可优化）

#### 2.1 风格轧差极值 → 策略适配度

**数据来源**：`size_spread/data/style_spread.json`

**核心指标**：
- 红利-科创、微盘-全指、中证2000-沪深300等轧差净值
- Z-score（当前值 vs 60日均值和标准差）

**整合方案**：
```python
# 在 macro_score.py 增加风格轮动信号
def get_style_rotation_signal():
    """从风格轧差判断轮动机会"""
    signals = []
    
    spread_data = load("size_spread/data/style_spread.json")
    if spread_data:
        for pair_name, data in spread_data.items():
            navs = data.get('navs', [])
            if len(navs) >= 60:
                recent = navs[-1]
                mean_60 = sum(navs[-60:]) / 60
                std_60 = statistics.stdev(navs[-60:])
                
                z_score = (recent - mean_60) / std_60 if std_60 > 0 else 0
                
                if abs(z_score) > 2:  # 偏离2倍标准差
                    direction = "做多" if z_score < -2 else "做空"
                    signals.append({
                        'pair': pair_name,
                        'z_score': round(z_score, 2),
                        'direction': direction,
                        'strength': min(abs(z_score) / 3 * 100, 100)
                    })
    
    return signals

# 在 fit_quant_stock() 中使用
def fit_quant_stock(config):
    score = 50
    signals = []
    # ... 原有逻辑 ...
    
    # 新增：风格轮动机会
    rotation = get_style_rotation_signal()
    for r in rotation:
        if '中证2000-沪深300' in r['pair'] or '微盘-全指' in r['pair']:
            score += r['strength'] * 0.15  # 最多加15分
            signals.append(f"{r['pair']} Z={r['z_score']:.1f}，{r['direction']}机会 🟢")
    
    return {"score": clamp(score), "signals": signals}
```

**预期效果**：
- 识别风格轮动的极值机会
- 提升量化股票和套利策略的适配度判断
- 捕捉均值回归交易机会

#### 2.2 宏观打分的趋势权重

**问题**：当前只看绝对水平和短期变化，缺少趋势持续性判断

**整合方案**：
```python
# 在 score_fundamentals() 中增加趋势权重
def score_fundamentals(d):
    score = 50
    # ... 原有逻辑 ...
    
    # 新增：趋势持续性加权
    pmi_list = d.get("pmi", [])
    if len(pmi_list) >= 6:
        vals = [x["pmi"] for x in pmi_list[-6:] if x.get("pmi")]
        if len(vals) >= 6:
            # 计算6个月线性回归斜率
            x = list(range(len(vals)))
            slope = (sum((i - sum(x)/len(x)) * (v - sum(vals)/len(vals)) 
                        for i, v in zip(x, vals)) /
                    sum((i - sum(x)/len(x))**2 for i in x))
            
            if slope > 0.5:  # 持续改善
                score += 15
                signals.append("PMI持续改善趋势 🟢")
            elif slope < -0.5:  # 持续恶化
                score -= 15
                signals.append("PMI持续恶化趋势 🔴")
    
    return clamp(score)
```

**预期效果**：
- 区分"持续改善"和"单月反弹"
- 提升宏观打分的前瞻性
- 减少噪音信号

**目标**：用动态阈值替代绝对阈值

**实现方案**：

# 保存历史风险分数到 alerts_history.json
# 每次计算时加载历史，计算分位
def calc_risk_percentile(current_score, history_scores):
    """计算当前风险分数在历史中的分位"""
    if len(history_scores) < 60:
        return 50
    return (sum(1 for s in history_scores if s < current_score) / 
            len(history_scores)) * 100

## 关键洞察

### 1. 两套打分系统的互补性
- **风险预警**：自下而上，从微观指标识别极端风险
- **宏观打分**：自上而下，从宏观环境评估策略适配度
- 两者结合才能形成完整的风险-收益评估框架

### 2. Meme反身性是当前最大盲区
- 传统指标（估值/流动性/情绪）在Meme行情中失效
- 叙事联动+量能加速度是识别反身性高峰的关键
- 这是2021-2024年市场最重要的结构性变化

### 3. 耐心资本成本是资金流的"第二层"
- 当前系统只看资金流方向（流入/流出）
- 但主力持仓成本决定了"愿不愿意卖"
- 成本偏离度是判断反弹/抛压的关键

### 4. 风格轮动是alpha的重要来源
- 风格轧差到极值 → 均值回归机会
- 当前系统缺少这个维度
- 整合后可提升量化股票和套利策略的适配度判断

### 5. 趋势持续性比绝对水平更重要
- PMI连续3个月上行 vs PMI单月反弹，前者信号更强
- 当前系统对两者评分相同
- 增加趋势斜率可提升前瞻性
