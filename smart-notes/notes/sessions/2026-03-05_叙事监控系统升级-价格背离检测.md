## 问题1：叙事生命周期阶段判断（待升级）

### 当前版本的不足

`get_lifecycle()` 只看7天评分曲线形状（连续上升/高位平稳），没有：
- 历史分位（这条叙事在过去60天的相对强度）
- 价格联动（叙事热时价格有没有同步）
- 持续时长（这条叙事热了多少天）

### 升级方案

```
叙事阶段 =
  评分曲线形状（近7天）
  + 历史分位（当前分 vs 过去60天分布）
  + 价格是否跟上
  + 持续时长

萌芽：分位<40% + 快速上升 + 价格未动
狂热：分位>80% + 持续≥5天 + 价格同步大涨
饱和：分位>80% + 持续≥10天 + 价格涨幅趋缓
衰退：从高位下降 + 价格开始回落
换皮：分数降但传导链二级资产开始动
```

## 问题3：下一步预测（待做）

### 已有配置文件

- `narrative_evolution.json` — 叙事换皮路径（e.g. 黄金：通胀hedge→去美元化→地缘避险→货币贬值）
- `transmission_chain.json` — 跨资产传导链（一级0-1周 / 二级1-4周 / 三级4-12周）

### 实现思路

当叙事处于**饱和/衰退期**时，自动查找 `narrative_evolution.json` 中的下一阶段，并从 `transmission_chain.json` 找出尚未price in的二级/三级资产，作为"换皮机会"推送。
