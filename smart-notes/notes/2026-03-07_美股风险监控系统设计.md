## 背景：为什么需要美股风险监控

Roni 对当前美股崩盘风险的判断：

**看空逻辑链条：**
1. **地缘风险**：伊以战争短期解决不了 → 持续不确定性
2. **美国基本面**：不太行 → 盈利预期下调
3. **AI叙事理性化**：从狂热转向理性 → 泡沫降温信号
4. **Capex-需求错配**：大厂疯狂投AI基础设施，但实际付费客户没那么多 → 投资回报率质疑

**核心矛盾：** AI Capex-需求错配是当前市场最大争议点。如果这个逻辑被市场接受，科技股会大幅回调。

## 核心设计哲学：不打分，看趋势

### 为什么不做固定权重打分？

Roni 的洞察：
> "市场或者整个世界我看不懂，所以逻辑上来说固定的打分并不能帮助我们思考，哪个变量变了权重就得改变"

**问题：**
- 2020年疫情时，流动性最重要（美联储印钱）
- 2022年通胀时，利率最重要（美联储加息）
- 2023年AI热潮时，估值泡沫最重要（科技股狂飙）
- 现在担心的是：AI Capex错配 + 地缘风险

**固定权重的缺陷：**
- 刻舟求剑，无法适应环境变化
- 掩盖真实信号（某个指标可能很重要，但权重低）
- 让人懒于思考（看到60分就觉得"中等风险"，不去深究）

### 新的设计方向

**核心：原始数据 + 自主判断**

不是"告诉你风险多少"，而是"帮你更好地思考"：
- 提供数据，不提供结论
- 展示趋势，不预设权重
- 辅助判断，不替代判断

用户根据：
- 当下的宏观环境
- 自己的投资组合
- 自己的风险偏好
- 自己的经验直觉

来决定哪些指标更重要，如何组合判断。

## 系统设计

### 数据架构

**时间范围：** 2008年至今（约18年，覆盖3次危机）

**数据源：**
- yfinance：VIX、标普、纳指、黄金、美元、比特币、原油、美债（免费无限制）
- FRED API：利率、失业率、PMI（需免费key）

**存储：**
- CSV：原始数据缓存（cache/us_*.csv）
- JSON：前端展示（us_alerts.json）

**更新模式：** 增量更新（首次回填18年，后续只拉新数据）

### 核心指标（8个）

| 指标 | 类别 | 风险理解 |
|------|------|---------|
| VIX | 情绪 | 上升=恐慌增加，>30=极度恐慌 |
| 标普vs200MA | 技术 | 跌破=趋势转弱，<-10%=熊市 |
| 纳指vs200MA | 技术 | 跌破=科技股泡沫破裂风险 |
| 黄金 | 跨市场 | 上涨=避险情绪升温 |
| 美元指数 | 跨市场 | 飙升=全球避险+新兴市场压力 |
| 比特币 | 跨市场 | 暴跌=风险资产抛售 |
| WTI原油 | 跨市场 | >100=通胀压力+衰退风险 |
| 10年期美债 | 流动性 | >5%=融资成本高企，股市承压 |

### 危机事件标注

在数据中标注6个历史关键时点：
- 2008-09-15：雷曼倒闭
- 2008-10-10：VIX创纪录
- 2020-03-16：熔断
- 2020-03-23：市场底部
- 2022-03-16：首次加息
- 2022-06-15：加息75bp

**用途：** 帮助理解指标在历史危机中的表现，识别当前是否接近危机水平。

## 关键实现细节

### 增量更新逻辑

```python
def get_start_date(csv_path, default_days=6570):
    """确定起始日期（增量逻辑）"""
    last_date = get_csv_last_date(csv_path)
    if last_date is None:
        # 首次运行：回填18年
        return (datetime.now() - timedelta(days=default_days)).strftime('%Y-%m-%d')
    else:
        # 增量：从最后日期+1天开始
        return (last_date + timedelta(days=1)).strftime('%Y-%m-%d')
```

### 趋势判断

```python
def calc_trend(series, window=20):
    """计算趋势方向"""
    if len(series) < window:
        return "→"
    recent = series.tail(window)
    slope = np.polyfit(range(len(recent)), recent, 1)[0]
    if slope > 0.01:
        return "↗"
    elif slope < -0.01:
        return "↘"
    else:
        return "→"
```

### 风险等级

```python
def calc_level(current, thresholds):
    """判断风险等级"""
    if current >= thresholds.get('danger', float('inf')):
        return "🔴"
    elif current >= thresholds.get('warning', float('inf')):
        return "🟡"
    else:
        return "🟢"
```

## 投资哲学总结

**承认不确定性：** 市场太复杂，没有人能完全看懂

**反对机械化：** 固定权重的打分系统是刻舟求剑

**动态思考：** 不同时期，不同变量的重要性不同

**保持灵活：** 需要根据当下环境调整判断框架

**工具定位：** 不是告诉你答案，而是帮你更好地思考
