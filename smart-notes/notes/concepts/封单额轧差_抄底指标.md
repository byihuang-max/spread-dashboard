# 涨跌停封单额轧差 (Seal Spread) — A股抄底指标

> 2026-03-08 Roni × Musk 设计

## 核心逻辑

**封单额轧差** = 当日全市场涨停封单额合计 − 跌停封单额合计（亿元）

- **正值大** → 涨停封单远超跌停，多头封锁力量强，市场亢奋
- **负值大** → 跌停封单远超涨停，空头封锁力量强，恐慌蔓延
- **接近零** → 多空均衡

## 抄底信号

轧差在近 250 个交易日（约1年）做滚动分位排名：

| 分位数 | 信号 | 含义 |
|--------|------|------|
| <5% | 🟢 极度恐慌 | 跌停封单远超涨停，恐慌见底 → **抄底信号** |
| 5%~20% | 🔵 偏空 | 空头占优但未到极端 |
| 20%~80% | ⚪ 均衡 | 正常波动 |
| 80%~95% | 🟡 偏多 | 多头占优 |
| >95% | 🔴 极度亢奋 | 过热警告 |

## 历史验证

- 20260115: 轧差 **-320亿**，跌停封单 440亿 vs 涨停 120亿 → 🟢 极度恐慌
- 20260202: 轧差 **-651亿**，跌停封单 692亿 vs 涨停 41亿 → 🟢 极度恐慌
- 20250904: 轧差 **-36亿**，罕见反转日 → 🟢 极度恐慌

这些极端恐慌日后，市场大概率出现反弹。

## 辅助指标

- `seal_ratio`：涨停封单额占比（0~1），<0.3 极度恐慌，>0.9 极度亢奋
- `spread_ma5 / ma20`：均线平滑，连续走低 + 分位底部 = 高置信底部
- 可与涨停指数 BIAS、情绪周期交叉验证
