# 风格集中度

**定义：** 市场收益被少数风格因子（大小盘、行业、动量等）解释的程度。用主成分分析（PCA）前N个主成分的解释方差比衡量。

**与超额的关系：** 集中度高 → 市场被一两个风格主导（比如"只涨大票"）→ 量化多因子模型容易被单一风格碾压 → 超额难做。

**Roni的洞察：** "集中度和[[离散度]]是交互项，不能简单加权。" 两者组合才有意义：
- 高离散度 + 低集中度 = ✅ 最佳alpha环境
- 高离散度 + 高集中度 = ⚠️ 假象，分化来自风格而非个股alpha
- 低离散度 + 高集中度 = ❌ 最差环境
- 低离散度 + 低集中度 = 😐 平淡，超额一般
