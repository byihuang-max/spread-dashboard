# 强势股环境适配

**用途：** 判断当前市场环境是否适合强势股/打板策略。

**核心逻辑：** 强势股策略依赖赚钱效应和市场情绪，环境好时爆发力强，环境差时回撤也大。需要一个"油门/刹车"信号。

**模块位置：** `env_fit/momentum_stock/`

**数据构成：**
- 涨跌停家数、连板高度 → 赚钱效应
- 全A成交额 + CV → 资金活跃度和稳定性
- 北向资金 → 外资情绪参考

**关键讨论：**
- [[情绪量化]] — Roni提出的核心问题：情绪指标的阈值怎么定？
- 参数设计需要记录"为什么"，不只是"是什么"

**设计决策记录：**
- 成交额用20日均值+CV，而非简单均线 → 因为"稳定放量 > 脉冲放量"（Roni原话）
