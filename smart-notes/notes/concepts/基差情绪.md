# 基差情绪

**定义：** 股指期货相对现货的升贴水幅度，反映市场对冲需求和情绪。

**Roni的关键洞察（反直觉）：**
> "基差是结果变量不是原因。升水 = 情绪过热 + 大票虹吸 → 超额反而难做。"

这跟很多人的直觉相反 — 很多人以为升水=市场好=超额好做，但实际上：
- **深贴水** → 对冲成本高，但alpha环境往往好（恐慌时个股分化大）
- **升水** → 对冲有收益，但往往伴随风格极端集中（大票抱团）→ 超额反而收敛

**使用方式：** 必须用历史分位数比较，绝对值没有意义。当前基差在过去一年的分位决定信号强弱。

**在框架中的角色：** 修正项，不是核心驱动。先看[[离散度]]×[[风格集中度]]，再用基差情绪做修正。
