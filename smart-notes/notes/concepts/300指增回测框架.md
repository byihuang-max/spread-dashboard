# 300指增回测框架

**定义：** 沪深300指数增强策略的量化回测体系，目标是通过选股超配/低配成分股跑赢基准。

## 四步框架

### 1. 选股模型（Alpha模型）
- 输入：Alpha158因子（158个量价特征）
- 模型：LightGBM
- 输出：每只股票未来N天的预期收益率
- 训练方式：滚动训练（每隔几个月重新训练）

### 2. 组合构建（Portfolio Construction）
- 基准权重：沪深300成分股的市值加权
- 超配看好的（+2~5%），低配不看好的（-2~5%）
- 约束条件：
  - 行业偏离 ≤ ±3%
  - 个股偏离 ≤ ±3%
  - 跟踪误差 ≤ 5-8%
  - 满仓（不做仓位择时）

### 3. 换仓执行
- 频率：月度或双周
- 成本：佣金0.03% + 印花税0.05%(卖出) + 滑点0.1%
- 换手约束：单次≤30%

### 4. 绩效评价
- 超额收益 = 组合收益 - 沪深300收益
- 信息比率(IR) = 超额收益/跟踪误差（>1.5优秀）
- 超额最大回撤（<5%优秀）
- 月度胜率（>60%及格）

## 能力分级

| 级别 | 描述 | 需要的组件 |
|------|------|-----------|
| L1 | TopK选股+等权 | Alpha158 + LightGBM |
| L2 | 行业中性+跟踪误差 | + 行业分类 + 基准权重 |
| L3 | Barra优化器 | + CNE6风格因子 + CVXPY |
| L4 | 实盘（不做） | + 交易接口 |
