# 离散度

**定义：** 个股收益率的截面标准差，衡量市场中个股分化程度。

**与超额的关系：** 离散度高 → 个股分化大 → 选股alpha空间大 → 量化超额容易做。离散度低 → 个股同涨同跌 → 选谁都差不多 → 超额收敛。

**关键洞察：** 离散度不能单独看，必须和[[风格集中度]]交叉判断。高离散度+低集中度=最佳alpha环境；高离散度+高集中度=假象，alpha被风格beta吃掉。
