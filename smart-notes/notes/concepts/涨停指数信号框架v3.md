# 涨停指数信号框架 v3 — 三重交叉验证

> 2026-03-08 Roni × Eva 共同设计

## 核心问题

涨停指数（昨日涨停/首板等权买入，次日卖出的净值曲线）是**长期正期望**的策略。用固定 BIAS 阈值（如 ±3%）做拥挤判断，会导致信号在趋势行情中**一直报拥挤**，完全失去预警价值。

## Roni 的洞察

> "真正的降温是净值有回撤，且乖离率有变化，两者共同下降才是市场降温的过程。"

关键认知：
- **BIAS 高 ≠ 拥挤**。正期望曲线长期跑在均线上方是正常的
- **BIAS 变化方向**才有意义，而不是绝对值
- **净值回撤 + BIAS 同步下行** = 真实降温信号
- **净值上行 + BIAS 同步上行** = 真实升温信号

## 三重交叉验证框架

### 因子1：净值回撤

```
nav_drawdown = (当前净值 - 近20日峰值) / 近20日峰值 × 100
```

- 回撤 < -3% → 明显降温
- 回撤 < -1.5% → 边际降温
- 回撤 > -0.5% → 高位运行

### 因子2：BIAS 变化方向

```
bias_delta = 近3日BIAS均值 - 前3日BIAS均值
```

- Δ < -1.0 → BIAS 快速下行
- Δ < -0.5 → BIAS 边际下行
- Δ > +0.5 → BIAS 上行
- Δ > +1.0 → BIAS 加速上行

用3日均值平滑，避免单日噪声。

### 因子3：情绪周期标签

来自强势股情绪诊断系统（`momentum_sentiment`），综合涨停数、晋级率、炸板率、涨跌停比、封板质量五维指标：

- **退潮 / 冰点** → 情绪降温方向
- **加速 / 回暖** → 情绪升温方向
- **震荡 / 分歧** → 中性

## 共振信号判定

| 净值 | BIAS方向 | 情绪周期 | 信号 | 置信度 |
|------|----------|----------|------|--------|
| 回撤 | 下行 | 退潮/冰点 | 🔵 降温 | ★★★ |
| 回撤 | 下行 | 中性 | ⬇️ 边际降温 | ★★ |
| 高位 | 掉头下行 | 退潮 | 🟡 见顶退潮 | ★★★ |
| 高位 | 掉头下行 | 中性 | 🟡 见顶分歧 | ★★ |
| 止跌 | 回升 | 回暖/加速 | 🟢 筑底回暖 | ★★★ |
| 创新高 | 加速上行 | 加速 | 🔴 加速升温 | ★★★ |
| 上行 | 上行 | 回暖 | 🟠 持续升温 | ★★★ |
| 矛盾 | 矛盾 | — | 🟡 多空分歧 | — |
| 无方向 | 无方向 | 震荡 | ⚪ 中性 | — |

**置信度规则：**
- ★★★ = 三个因子方向一致
- ★★ = 两个因子一致，第三个中性
- 无星 = 信号矛盾或无方向

## 回测验证（120个交易日）

| 情绪周期 | 天数 | 平均BIASΔ | BIASΔ<0占比 | 涨停指数日收益<0占比 |
|----------|------|----------|------------|-------------------|
| **加速** | 30 | **+1.51** | 10% | 0% |
| 回暖 | 24 | +0.37 | 29% | 8% |
| 震荡 | 10 | -0.09 | 60% | 10% |
| 分歧 | 16 | +0.53 | 31% | 12% |
| **退潮** | 36 | **-1.60** | **75%** | **39%** |

结论：情绪周期与涨停指数BIAS方向**高度共振**，退潮日75%概率BIAS下行，加速日90%概率BIAS上行，交叉验证有效。

## 连续退潮才是真风险

单日退潮后，下一日大概率反弹（正期望曲线的均值回归）。但**连续退潮**（退潮→退潮）时风险才真正放大，例如：

- 20260302 退潮 → 20260304 退潮：涨停指数日收益 **-3.71%**，BIASΔ = **-6.05**

## 与旧逻辑对比

| | 旧逻辑 v1 | 新逻辑 v3 |
|--|-----------|-----------|
| 判据 | BIAS 绝对值 ±3% | 净值回撤 × BIAS方向 × 情绪周期 |
| 问题 | 正期望曲线下永远报拥挤 | 看变化方向和共振 |
| 信号类型 | 6种（固定阈值） | 8种（动态共振） |
| 置信度 | 无 | ★★★ / ★★ |
| 展示信息 | BIAS + 高低开 | BIAS + BIASΔ + 回撤% + 情绪周期 + 首板BIASΔ |
