# Barra CNE6 风格因子

**定义：** Barra CNE6 是 A 股指增的行业标准风险模型，把股票收益拆解为：行业因子 + 风格因子 + 残差alpha。指增的核心就是控制前两者，只赚残差alpha。

**核心公式：** `股票收益 = 行业暴露×行业因子收益 + 风格暴露×风格因子收益 + alpha`

## 十大风格因子

| 因子 | 英文 | 含义 | 不控的后果 | 需要的数据 |
|------|------|------|-----------|-----------|
| 市值 | Size | ln(总市值) | 大小盘轮动吃掉超额 | 总市值 |
| Beta | Beta | 对市场的敏感度 | 牛市虚胖、熊市裸泳 | 日收益率回归 |
| 动量 | Momentum | 过去252天收益(去近21天) | 风格反转时超额巨亏 | 日收益率 |
| 残差波动率 | Residual Vol | 特质波动(去市场后) | 组合波动大、不稳定 | 回归残差 |
| 非线性市值 | NL-Size | 中盘效应(Size³的残差) | 中盘暴露不可控 | 总市值 |
| 估值 | Book-to-Price | 1/PB | 价值/成长轮动吃超额 | 每股净资产 |
| 流动性 | Liquidity | 月均换手率 | 回测好看实盘跑不出 | 换手率 |
| 盈利 | Earnings Yield | 1/PE (EP) | 赌错盈利风格 | 净利润/市值 |
| 成长 | Growth | 营收/利润增长率 | 加息周期杀成长 | 财报数据 |
| 杠杆 | Leverage | 资产负债率 | 信用风险暴露 | 资产负债表 |

## 行业因子

30个申万一级行业，每只股票归属一个行业。指增要求行业偏离≤±3%，不做行业赌注。

## 关键洞察

- **Alpha158 vs Barra：** Alpha158是"进攻"（选股赚alpha），Barra是"防守"（控制风格暴露不被风格反转吃掉超额）
- **风格因子的本质：** 都是系统性风险。你可以赚，但也可以亏，而且不可控。指增只要确定能赚的alpha，不要不确定的风格beta
- **Size 是 A 股最大的坑：** 很多量化指增2021年超额爆表（小盘行情），2024年遇到小微盘暴跌超额回撤20%+——就是Size没控好
- **BP（价值/成长）在 GAMT 看板里有对应：** 风格轧差模块就是监控这个

## FOF经理评价量化指增时的核心问题

1. **风格暴露控制得怎么样？** → 看持仓的Barra因子暴露
2. **超额回撤时在赌什么风格？** → 归因到10个风格因子
3. **纯alpha有多少？** → 去掉风格贡献后的残差收益
4. **跟踪误差多大？** → 过高说明偏离基准太多
