# 行业财报风险系统升级为双轴评分体系

**日期**: 2026-04-06  
**类型**: 决策 / 框架升级  
**标签**: #财报风险 #行业规则 #双轴评分 #正负向 #万德三级行业 #传导链

---

## 核心决策

当前的"财报风险因子页面"不应只停留在"风险剔除系统"，而应升级为：

**分行业基本面双轴评分体系**

- **负向轴**：经营恶化分（风险剔除）
- **正向轴**：经营质量分（优等生识别）

这不是简单的"加一个反向打分"，而是整个系统定位的升级：

> 从"财报雷票识别器"  
> 升级为"分行业基本面评分与分层系统"

---

## 为什么可以正反一起做

一旦有了：
- **5 年全 A 股财务数据**（万德三级行业）
- 全市场覆盖
- 多年期面板
- 分行业可比
- 后验验证

这套系统就不只是"坏公司识别器"，而是：

**分行业的基本面质量评分系统**

它天然可以拆成两面：

### 负向端：差等生 / 风险生
- 哪些公司经营质量在恶化
- 哪些是财报雷
- 哪些该剔除
- 哪些是连续下台阶而不是短期波动

### 正向端：优等生 / 经营强者
- 哪些公司在本行业里经营质量持续领先
- 哪些不是题材强，而是基本面真强
- 哪些是利润、现金流、周转、负债表一起更健康
- 哪些行业里"最稳、最能打"的公司是谁

---

## 行业规则是整个系统的脊梁

行业规则不只是"这个行业看什么指标"，而是在回答 **5 个层次的问题**：

### 第 1 层：这个行业的生意本质是什么
- 赚的是周转的钱，还是杠杆的钱，还是研发兑现的钱
- 这决定了评分系统应该侧重哪个轴

### 第 2 层：什么叫"好"，什么叫"坏"
- 银行利润下滑 10% 和半导体利润下滑 10%，严重程度完全不一样
- 消费品库存高和电子库存高，含义完全不同
- 不能跨行业用统一阈值

### 第 3 层：恶化/改善的传导顺序
- **先行因子 → 同步验证 → 滞后结果**
- 有些行业是"应收先坏 → 现金流跟着坏 → 利润最后坏"
- 有些行业是"价格先跌 → 库存跌价 → 利润和现金一起坏"
- 不知道顺序，就不知道现在到底是"刚开始坏"还是"已经快坏完了"

### 第 4 层：什么情况下不该判负
- 半导体库存高但周转正常 → 战略备货
- 生物制药持续亏损但现金储备足 → 研发投入期
- 证券 CFO 为负 → 正常业务结构
- 这层决定了系统有没有"冤假错案"

### 第 5 层：什么情况下该判正
- 优等生的定义也必须分行业
- 消费品的优等生可能是"高周转 + 高现金创造"
- 科技股的优等生可能是"研发兑现 + 收入加速"
- 金融股的优等生可能是"资产质量稳 + 拨备厚"

## 升级后的行业规则结构

```
每个行业的规则 = {
  生意本质,
  
  负向轴: {
    bad_when,        // 什么叫差
    worse_when,      // 什么叫在变差
    not_worse_if,    // 什么情况不该判差
    传导链: [先行因子, 同步验证, 滞后结果],
  },
  
  正向轴: {
    good_when,       // 什么叫好
    better_when,     // 什么叫在变好
    not_better_if,   // 什么情况不该判好（假强）
    传导链: [先行因子, 同步验证, 滞后结果],
  },
  
  阈值: {
    行业内分位基准,   // 用 5 年全量数据算出来
    极端值处理,       // 金融类/周期类需要特殊处理
  },
  
  特例二级行业: [
    { 名称, 归属, 差异点, 独立规则 }
  ]
}
```

## 传导链的核心价值

**传导链不是装饰，而是这套系统的核心竞争力。**

它回答的是：

### 对于负向轴
- 这个行业的公司，通常是从哪里开始坏的？
- 什么指标是"先行信号"（最早反映问题）
- 什么指标是"同步验证"（确认恶化在传导）
- 什么指标是"滞后结果"（已经坏完了）

### 对于正向轴
- 这个行业的优等生，通常是从哪里开始强的？
- 什么指标是"先行信号"（最早反映改善）
- 什么指标是"同步验证"（确认质量在提升）
- 什么指标是"滞后结果"（已经变成稳定优势）

### 为什么传导链很重要
因为它决定了：
- **你现在看到的信号，处于恶化/改善的哪个阶段**
- 是刚开始坏，还是已经快坏完了
- 是刚开始好，还是已经进入稳定期

这直接影响 PM 的决策：
- 如果只看到先行因子恶化，可能还有时间观察
- 如果同步验证也开始恶化，说明传导链已经启动
- 如果滞后结果也恶化了，说明已经坏透了

## 最终框架：双轴评分 + 行业内分层

### 输出 1：负向轴（经营恶化分）
- 谁在变坏
- 适合：剔除项、风险预警、雷票识别

### 输出 2：正向轴（经营质量分）
- 谁在变好 / 谁长期更强
- 适合：优等生筛选、行业龙头质量比较、基本面优选池

### 输出 3：行业内分层
- 优等生
- 稳健生
- 普通生
- 风险生
- 雷区生

### 输出 4：风险-质量二维图
横轴：经营质量  
纵轴：恶化风险

公司会分成四象限：
- **右下：高质量低风险** → 真优等生
- **右上：高质量高风险** → 可能景气拐点/高波动成长股
- **左下：低质量低风险** → 平庸但暂时稳定
- **左上：低质量高风险** → 真正该规避

## 价值与目标

### 对 PM 的价值
- **正向**：识别行业内真正的优等生，不只看题材和情绪
- **负向**：识别财报雷和经营恶化股，做风险剔除
- **多空**：正负两端都能用，甚至可以做多空组合

### 对行业配置的价值
- 行业内优等生占比
- 行业内风险生占比
- 行业整体基本面质量趋势
- 行业 ETF 的基本面支撑强弱

### 最终定位
这不是一个"财报雷票页面"，而是：

**分行业基本面评分与分层系统**

它有两套标准：
1. **打分体系**：正向质量分 + 负向风险分
2. **财务体系**：分行业的财务恶化/改善传导链
