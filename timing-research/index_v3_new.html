✅ 为什么封单额轧差更强

1. 直接刻画资金力量：不只看家数，看资金堆在哪边
2. 信号质量高：胜率70-90%，远超家数/比例的50-60%
3. 回撤控制好：最大回撤1-6%，风险可控
4. 跨指数稳定：300/500/1000/2000都有效

🎯 核心结论

1. 核心信号：封单额轧差（seal_diff）
2. 最优标的：沪深300（胜率最高，回撤最小）
3. 最优参数：5%阈值（极端冰点），10天持有
4. 信号特征：稀缺但质量极高（7年12次交易）

🎯 风格调优：在封单额轧差基础上叠加市值风格判断

基础策略：封单额轧差触发冰点 → 默认做沪深300（最稳）
风格增强：如果当前是小盘风格主导 → 切换到中证2000（增强收益）

📊 风格判断逻辑

用中证2000 / 沪深300相对强弱判断风格：
• 相对强弱 > 20日均线 → small（小盘风格）
• 相对强弱 < 20日均线 → large（大盘风格）
• 其他 → neutral（中性）
