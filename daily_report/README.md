# daily_report — FOF 每日市场观察报告

## 概述

一键生成 A 股 + 全球市场日报文本，覆盖宏观、中观、微观三个层面，自动输出到 `/tmp/fof_report.txt`，可手动推送到飞书/Telegram。

## 文件说明

| 文件 | 用途 |
|------|------|
| `fof每日市场飞书脚本v3.py` | 当前主用脚本（v3），跑一次出完整日报 |

## 数据源

双源架构，互为补充：

| 数据源 | 覆盖范围 | 接口 |
|--------|----------|------|
| Tushare | A股深度数据（宽基指数、成交额、涨跌停、北向资金、两融、宏观经济指标、期货） | HTTP POST 到私有服务器 |
| iFind | 全球实时行情（纳指100、费城半导体、恒生/恒生科技、USDCNY/CNH） | REST API（需 access_token） |

沪深300 和 USDCNH 两个数据源都能拿到，做交叉校验。

## 报告内容

1. 美债利率（2Y/10Y/10Y-2Y利差）
2. 汇率（USDCNY/USDCNH）
3. Shibor（隔夜/1周/1月/3月）
4. 全球指数（纳指100、费城半导体、恒生指数、恒生科技）
5. 宏观经济（CPI/PPI/PMI/M1/M2，最新值+前值+变化）
6. 期货（螺纹钢、铁矿石、玻璃、纯碱、铝，当日涨跌+20日趋势）
7. A股8大宽基指数（沪深300/上证50/中证500/1000/2000/科创50/创业板/北证50）
8. 全A成交额 + 近3年分位数
9. 涨跌家数、涨跌停统计
10. 北向资金（当日/5日/20日累计）
11. 两融余额（MA5）
12. 自动结论 + 红灯预警事件清单

## 运行

```bash
cd ~/Desktop/gamt-dashboard/daily_report
python3 fof每日市场飞书脚本v3.py
```

输出到 `/tmp/fof_report.txt`。
