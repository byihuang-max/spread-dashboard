## 一句话解释

这个看板用来回答一个问题：**现在市场有没有在走 Meme 行情？走到哪个阶段了？**

---

## 理论背景

**索罗斯的反身性理论（Reflexivity）**

正常市场：基本面 → 价格  
Meme 市场：**叙事 → 价格 → 强化叙事 → 更高价格**（正反馈循环）

Meme 行情不是"被高估的泡沫"，而是**叙事本身成为基本面**。有多少人相信这个故事，比公司值多少钱更重要。

这个循环有四个阶段：

| 阶段 | 信号 | 策略 |
|------|------|------|
| 🟢 叙事离散期 | 各资产走独立行情，无主导叙事 | 选股/选策略，不做方向博弈 |
| 🟡 叙事聚合期 | 各资产开始联动，叙事在形成 | 观察方向，少量试仓 |
| 🟠 反身性强化期 | 正反馈循环建立，量能放大 | 可参与趋势，控制仓位，设止损 |
| 🔴 反身性高峰期 | 极度同步，量能可能开始衰竭 | 注意尾部风险，警惕循环破裂 |

## 两个核心指标

### 叙事联动指数（NLI, Narrative Linkage Index）

**来源：** 30天滚动相关性矩阵（`rolling_corr.json`）

**算法：**
```
取相关性矩阵中所有"正相关"的非对角线值，求平均
```
- 正相关 = 资产同涨同跌（叙事联动）
- 负相关 = 资产分化（不算联动，排除）
- 对角线 = 自己和自己的相关性（=1，排除）

**解读：**
- 高 → 各资产同涨同跌，同一叙事驱动市场 → Meme 行情信号
- 低 → 各资产走独立行情 → 无主导叙事
- ⚠️ 恐慌暴跌时也会升高（全面崩跌也是"联动"），需结合价格方向判断

---

### 量能加速度（VA, Volume Acceleration）

**来源：** `antifragile_nav.json` 中的 `vol_data`

**算法：**
```
对每个资产：VA = 近5日均量 / 近30日均量 - 1
然后按权重加权平均：
  BTC          35%  ← Crypto最先感知叙事情绪，是领先指标
  纳斯达克ETF  25%  ← 美国科技，全球Meme主战场
  恒生科技ETF  20%  ← 港股科技
  科创50ETF   20%  ← A股科技
```

**解读：**
- `> +30%` → 量能爆发，参与者激增，叙事在快速扩散
- `0 ~ +30%` → 量能温和放大，健康趋势
- `< 0` → 量能萎缩，即使价格还在涨也是顶部警惕信号（**量价背离**）

---

### 综合 Meme 信号（0-100分）

```
综合信号 = NLI历史分位（0-100）× 50% + VA历史分位（0-100）× 50%
```

**历史分位：** 用过去252个交易日（≈1年）作为样本窗口。
分位90表示：今天的信号，比过去1年中90%的日子都更强。

## 常见问题

**Q: 为什么相关性矩阵颜色全红是Meme信号？**
A: 红=正相关（同涨同跌）。全红=所有资产叙事趋同=Meme行情。蓝=负相关（分化）。

**Q: NLI高但价格在跌，怎么理解？**
A: 可能是系统性恐慌（全面崩跌也是"联动"）。结合VA判断：
- VA高 + NLI高 + 价格涨 → Meme狂欢
- VA高 + NLI高 + 价格跌 → 恐慌清算
- VA低 + NLI高 + 价格涨 → ⚠️量价背离，顶部信号

**Q: QQQ的量和^NDX的量一样吗？**
A: 不完全一样，但高度相关。QQQ是追踪纳指100最大的ETF，成交量是全球最流动的ETF之一，可以作为纳指热度的代理。

**Q: 数据多久更新一次？**
A: 手动运行一次管线就更新一次。可以设定每天收盘后（15:35或美股收盘后）自动跑。
